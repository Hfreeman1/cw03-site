Exercise 5 – School Website (CSS Methods)

1) Advising the School

Use one shared external stylesheet for all pages. Keep a simple style guide (colors, fonts, headings). Ask teachers to use the provided HTML templates and have one person review pages before posting.

2) Process

Start with a base external CSS file for the whole site. If a page needs a small change, use a short internal style on that page. Use inline styles only for one-time, special cases. This keeps the look consistent and makes updates easier.

3) HTML Tags and CSS Styles

Examples of tags (shown as text using entities):

<header>, <nav>, <main>, <section>, <article>, <footer>, <h1>–<h6>, <p>, <ul>, <ol>, <li>, <a>, <table>, <tr>, <th>, <td>, <div>, <span>

Useful CSS: color, background, font-family, font-size, font-weight, margin, padding, border, border-radius, display, width, text-align.
